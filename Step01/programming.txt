고급언어 : 사람이 알기 쉬운 언어로 작성된 코드
	-> C, C++, JAVA, C#, js, python.......
저급언어 : 기계쪽에 가까운 언어(기계가 알아보기 쉬운 언어)
	-> Assembly
	
	코드(소스코드) --> 원시 프로그램
		|
		|   번역(컴파일) --> 문법 체크
		▼
	목적 프로그램  --> 기계가 알아볼수있는 코드로 변환
		|
		|	빌드(링커) --> 목적 프로그램들을 묶어서 실행 파일로 만듬
		▼
	실행 프로그램
		|
		|	로더 --> 프로그램을 메인 메모리에 적재
		▼
	  프로세스
	  
컴파일(Compile) : 작성한 소스코드를 처음부터 끝까지 번역 --> 목적프로그램이 결과물
	번역을 해주는 프로그램을 컴파일러(Compiler)
	--> C, C++, Java, C#
	
인터프리터(Interpreter) : 코드를 한줄씩 읽어서 해석해서 실행
	--> javascript, python.....


절차지향언어(procedural programming) : 작성된 코드 순서대로 실행하는 언어(작업해야될 순서)
		--> C, Assembly

객체지향언어(Object Oriented Programming) : 일을 하는 대상을 표현하는 언어
		--> C++, Java, C#, javascript, python
		
1. 출력
2. 데이터 저장
3. 입력
4. 연산자
5. 반복문
6. 분기문
